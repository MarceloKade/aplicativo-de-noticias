### Apresentação

Bem-vindo ao Aplicativo de Notícias! Este projeto é um aplicativo web desenvolvido utilizando o framework Next.js.

### Objetivo

O objetivo deste projeto é criar uma aplicação de notícias que exibe as principais manchetes de diferentes categorias. Além disso, o aplicativo permite aos usuários ler parte do conteúdo do artigo da notícias em um modal e oferece uma experiência responsiva para dispositivos móveis e desktops.

### Funcionalidades

- [x] Exibir as principais manchetes de notícias em destaque.
- [x] Permitir aos usuários selecionar uma categoria específica de notícias, como negócios, técnologia, saúde, etc.
- [x] Ao clicar em uma notícia, abre um modal com o conteúdo completo da notícia.
- [x] Responsivo, adequado para visualização em dispositivos móveis e desktops.

### Tecnologias

As seguintes tecnologias foram utilizadas neste projeto:

- Next.js.
- TypeScript.
- Tailwind CSS.
- Axios.
- React Modal.

### O que aprendi
Durante o desenvolvimento deste projeto, aprendi diversas habilidades e conceitos importantes. Aqui estão algumas das principais lições que obtive:

- **Criação de Modal:** Aprendi a criar um modal personalizado para exibir conteúdo de forma elegante e interativa. Com a implementação do modal, posso exibir informações adicionais, como um título, o conteúdo da notícia, a data de publicação, uma imagem e uma fonte relacionada. Essa funcionalidade me permite criar uma experiência mais agradável para o usuário, onde ele pode interagir com o modal para visualizar detalhes adicionais conforme necessário.
- **Utilização de media queries com Tailwind CSS:** Também aprendi a utilizar media queries em conjunto com o poderoso framework Tailwind CSS. Com o uso de media queries, posso criar estilos responsivos que se adaptam a diferentes tamanhos de tela e dispositivos móveis. O Tailwind CSS facilita a aplicação de classes responsivas, permitindo que eu ajuste facilmente o layout, o espaçamento, as fontes e outros estilos com base nas necessidades de cada dispositivo. Dessa forma, posso garantir que meus modais e outros componentes sejam visualmente agradáveis e funcionais em diferentes telas, proporcionando uma experiência consistente em todos os dispositivos.
